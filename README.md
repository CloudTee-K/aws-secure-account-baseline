# AWS Secure Account Baseline (Lab 0)

## Objective 

Build a secure AWS account foundation using industry best practices to prevent common cloud security failures.
Hands-on AWS Cloud Security Project: secure account foundation using IAM, CloudTrail and AWS Config

## Scope
This lab focuses on:
- Root account protection
- IAM hardening
- Centralized audit logging
- Configuration monitoring

## Security Controls Implemented
- Root account MFA enforcement
- IAM administrative user with MFA
- Strong password policy
- CloudTrail enabled for all regions
- AWS Config enabled for configuration drift detection

## Threats Mitigated
- Root credential compromise
- Unauthorized API activity
- Privilege abuse
- Undetected configuration changes

## Tools & Services
- AWS IAM
- AWS CloudTrail
- AWS Config
